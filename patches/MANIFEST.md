## Applying the patches to a fresh vendor drop

```sh
# 1. Get the vendor tarball
curl -O http://heirloom.sourceforge.net/... /heirloom-devtools-070527.tar.bz2
tar xjf heirloom-devtools-070527.tar.bz2
cd heirloom-devtools-070527

# 2. Initialise git for patch application
git init && git add -A && git commit -m "vendor: pristine heirloom-devtools-070527"

# 3. Apply the patches in order
git am /path/to/heirloom-devtools-darwin/patches/*.patch
```

Or as a single diff (loses per-commit metadata but works for
systems without `git am`):

```sh
cd heirloom-devtools-070527
patch -p1 < /path/to/heirloom-devtools-darwin/patches/cumulative.diff
```
